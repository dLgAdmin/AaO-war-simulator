# ⚔️ AaO War Simulator

**Total Battle** tarzı strateji oyunları için geliştirilmiş, mobil uyumlu **savaş ve ordu dizilim hesaplayıcısı**. Liderlik ve nüfuz kapasitene göre en verimli birlik ve canavar dağılımını saniyeler içinde hesaplar.

🔗 **Canlı sürüm:** [Siteye git](https://aaohelper.com)

## ✨ Özellikler

- 🛡️ **Muhafız ve Özel birlikler** — seviye seçimli (I–VII) dizilim hesaplama
- 🐉 **M3–M9 canavar desteği** — güç, sağlık ve nüfuz değerleriyle
- 📊 Liderlik kapasitesine göre otomatik oran dağılımı (uçan / süvari / piyade / menzilli)
- 🌍 **8 dil desteği:** Türkçe, İngilizce, Almanca, Fransızca, İspanyolca, Rusça, Portekizce, Arapça (RTL dahil)
- 📋 Sonuçları tek dokunuşla kopyalama ve paylaşma
- 📱 Tamamen mobil uyumlu, tek HTML dosyası — kurulum gerektirmez

## 🚀 Kullanım

1. Siteyi aç
2. Liderlik ve nüfuz kapasiteni gir
3. Birlik seviyelerini ve canavarlarını seç
4. **Hesapla** — ordu dizilimin hazır!

## 🛠️ Teknoloji

Saf **HTML + CSS + JavaScript**. Harici bağımlılık yok, çevrimdışı da çalışır.
